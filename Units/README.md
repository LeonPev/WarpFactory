
# Units Module Documentation


## Unit Functions

These functions define conversion factors for various units.

* **cm.m:** Centimeters.
* **gram.m:** Grams.
* **kg.m:** Kilograms.
* **km.m:** Kilometers.
* **meter.m:** Meters.
* **mm.m:** Millimeters.
* **ms.m:** Milliseconds.
* **tonne.m:** Tonnes.
* **second.m:** Seconds.

## Universal Constants

* **G.m:** Gravitational constant.
* **c.m:** Speed of light.
